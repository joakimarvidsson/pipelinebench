# Security

PipelineBench is designed for local, offline benchmark runs. It is not a sandbox.

Validation commands execute as normal local subprocesses from the task workspace. Review community
tasks before running them, especially starter code, tests, and task metadata. If you need stronger
isolation, run PipelineBench inside a container or disposable development environment.

Please report security concerns through GitHub security advisories once the public repository is
available. Do not disclose vulnerabilities publicly before maintainers have had a chance to respond.
